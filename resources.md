# Resources for Urban Data Skills

## Australian Data Sources

### Australian Bureau of Statistics (ABS)
- **Census Data**: Comprehensive demographic and housing data from the Australian Census
  - [ABS Census](https://www.abs.gov.au/census)
  - Available at various geographic levels (SA1, SA2, LGA, etc.)
- **ABS Data by Region**: Interactive data exploration tools
  - [Data by Region](https://dbr.abs.gov.au/)
- **TableBuilder**: Create custom tables from ABS microdata
- **ABS Maps**: Spatial visualization of statistical data

### State and Territory Data Portals
- **Victoria**: [DataVic](https://www.data.vic.gov.au/)
- **New South Wales**: [NSW Data.NSW](https://data.nsw.gov.au/)
- **Queensland**: [Queensland Government Data](https://www.data.qld.gov.au/)
- **South Australia**: [data.sa](https://data.sa.gov.au/)
- **Western Australia**: [WA Data](https://data.wa.gov.au/)
- **Tasmania**: [Tasmanian Data Portal](https://data.gov.au/organization/tas-gov-au)
- **Australian Capital Territory**: [ACT Open Data Portal](https://www.data.act.gov.au/)
- **Northern Territory**: [NT Data Portal](https://data.nt.gov.au/)

### Spatial and Planning Data
- **PSMA Australia**: Geocoded National Address File (G-NAF) and other spatial datasets
- **Geoscience Australia**: Topographic and geological data
- **AURIN**: Australian Urban Research Infrastructure Network - comprehensive urban datasets
- **data.gov.au**: Australian Government open data portal

### Transport and Infrastructure
- **Bureau of Infrastructure and Transport Research Economics (BITRE)**: Transport statistics and analysis
- **State transport agencies**: Public transport, traffic, and infrastructure data

### Environmental Data
- **Bureau of Meteorology**: Climate and weather data
- **SEED Portal**: NSW environmental data
- **Clean Energy Regulator**: Energy and emissions data

## Learning Resources

### Data Skills Development
- **ABS Learning Resources**: Tutorials and guides for using ABS data
- **AURIN Training**: Workshops and resources for using urban datasets
- **Data.gov.au Academy**: Training on open data use

### GIS and Spatial Analysis
- **QGIS**: Free and open-source GIS software
  - [QGIS Training Manual](https://docs.qgis.org/latest/en/docs/training_manual/)
- **ArcGIS**: Commercial GIS platform (institutional licenses common)
  - [Esri Training](https://www.esri.com/training/)

### Data Analysis Tools
- **R**: Statistical programming language
  - [R for Data Science](https://r4ds.had.co.nz/)
  - Packages: `sf`, `tidyverse`, `ggplot2`
- **Python**: General-purpose programming with strong data libraries
  - [Python for Data Analysis](https://wesmckinney.com/book/)
  - Libraries: `pandas`, `geopandas`, `matplotlib`
- **Excel**: Widely used for basic data analysis
  - Power Query and Power Pivot for advanced work

### Data Visualization
- **Tableau Public**: Free data visualization tool
- **Power BI**: Microsoft's business intelligence platform
- **R Shiny**: Interactive web applications from R
- **Python Dash**: Interactive dashboards in Python

## Glossary of Terms

**ABS**: Australian Bureau of Statistics - Australia's national statistical agency

**AURIN**: Australian Urban Research Infrastructure Network - provides access to urban datasets

**GIS**: Geographic Information System - software for spatial data analysis

**LGA**: Local Government Area - administrative boundary for local councils

**SA1, SA2, SA3, SA4**: Statistical Areas at different levels of aggregation (ABS geography)

**Open Data**: Data that is freely available for use and redistribution

**API**: Application Programming Interface - programmatic access to data
